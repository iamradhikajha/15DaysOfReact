Axios is a JavaScript library used to make HTTP requests from the browser or Node.js.
It helps React applications communicate with backend servers and APIs easily.

### 2Ô∏è‚É£ Axios vs Fetch

Axios is easier to use than the Fetch API.
It automatically converts data to JSON, handles errors better, and has cleaner syntax.

### 3Ô∏è‚É£ Installing Axios

Axios is installed using npm and added to the React project so it can be used for making API calls.

### 4Ô∏è‚É£ Making GET Requests

Using Axios, we can send a GET request to a backend or public API to fetch data.

### 5Ô∏è‚É£ useState with Axios

The useState hook is used to store the data received from the API.
This allows React to update the UI whenever new data is fetched.

### 6Ô∏è‚É£ Asynchronous API Calls

Axios works asynchronously, meaning the app does not freeze while waiting for data from the server.

### 7Ô∏è‚É£ Rendering API Data in UI

The data received from the API is displayed on the screen using .map() inside JSX.

### 8Ô∏è‚É£ Button-Based Data Fetching

A button click triggers the Axios request, allowing the user to fetch data when needed.

### 9Ô∏è‚É£ Connecting Frontend with Backend

Axios acts as a bridge between the React frontend and the backend API.

### üîü Handling API Response
